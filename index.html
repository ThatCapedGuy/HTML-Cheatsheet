Semantic HTML Reference

Tags

Tag Name Description
<header> Header Contains information at the top of the webpage, such as a title or a logo
<main> Main Contains the main content of the webpage
<footer> Footer Contains content that goes at the bottom of the webpage
<nav> Navigation Contains links to the content of a webpage; a table of contents
<video> Video Embeds a video file into the webpage
<audio> Audio Embeds an audio file into the webpage
<figure> Figure Encloses a piece of media such as a table or an image
<figcaption> Figure Caption Describes a piece of media enclosed in the <figure> element
<section> Section Describes content within a webpage that all has the same theme
<article> Article Describes a large body of content that stands on its own such as a blog entry or a news article
<aside> Aside Contains content that is related to the main body of content but not necessary for understanding it
